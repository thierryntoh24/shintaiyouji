## Libraries & data sources

This project stands on the shoulders of:

- **Jean Meeus** — *Astronomical Algorithms*, 2nd ed. (1998). The foundation for every solar and lunar calculation in this app.
- **NOAA Solar Position Algorithm** — Equation of Time, sunrise/sunset methodology.
- **OpenStreetMap / Nominatim** — geocoding. Free, open, and quietly keeping the internet located.
- **Open-Meteo** — weather data. No API key, no nonsense.
- **TimeAPI** — timezone resolution. Falls back to TimeZoneDB.
- **MapTiler** — map tiles and globe rendering.
- **MapLibre GL** — the open-source WebGL map renderer.
- **shadcn/ui** — components.
- **Tailwind CSS** — styling.
- **Next.js** — framework.
